# 歌词结构详解

## 三段论框架

### 🎼 Verse (世界观铺陈) - 约 8-12 行

**功能**: 建立场景，埋下伏笔

**关键技巧**:
- **特写镜头**: 不写"世界很冷"，写"霓虹灯影里，铁锈的味道在扩散"
- **感官细节**: 加入声音、气味、触觉描述
- **意象前置**: 核心意象首次出现

**示例结构**:
```
[意象 1 特写] 破碎的法环在月光下闪烁
[动作描写] 林夜指尖轻触，冰冷的契约纹浮现
[环境烘托] 深渊的呼吸已淹没街角的灯火
[内心矛盾] 是继续沉沦，还是逆流而上？
```

### ⚡ Pre-Chorus (情绪攀升) - 约 4-6 行

**功能**: 积蓄张力，推动转折

**关键技巧**:
- **疑问句式**: "谁在深渊边缘呼唤？"
- **动作短语**: 制造紧迫感
- **节奏加速**: 句长逐渐缩短

**示例结构**:
```
[突变] 契约失控的瞬间，恶魔在瞳孔中低语
[抉择] 退路已断，唯有向前
[决心] "这一战，我命由我不由天"
```

### 🎯 Chorus (主题爆发) - 约 8-10 行 (需押韵)

**功能**: 全曲核心，点明主旨

**关键技巧**:
- **严格押韵**: 每行末尾字同韵
- **字数对齐**: 推荐 10±2 字/行
- **核心口号**: 关于命运、反抗、希望
- **收尾扣题**: 呼应小说标题

**示例结构**:
```
[核心口号] 逆转这宿命的荒唐
[核心意象] 在废墟之上寻找光芒
[口号 2] 哪怕世界已走向灭亡
[意象 2] 契约书染血却仍在发烫
[收尾] 深渊尽头，我是唯一的希望
```

## 📝 完整示例

### 《深渊契约师》- 虐向燃向转场

```
[Verse 1]
月光下，法环碎裂成光尘
染血的契约书，在掌心发烫
街灯下，恶魔低语如旧友
林夜转身，背影被夜色拉长

[Pre-Chorus]
当力量即将吞噬最后的理智
当人性在深渊边缘摇摇欲坠
是沉沦，还是反击这荒谬
谁在呼唤，谁在等待答案

[Chorus]
契约在燃烧，燃尽所有伪装
深渊在咆哮，呼唤着希望之光
哪怕被诅咒，哪怕被世界遗忘
我仍坚守，最后一丝人性的回响

[Verse 2]
破碎的月，映照扭曲的脸庞
触手缠绕，却束缚不了倔强
每一次呼吸，都在与恶魔较量
每一次抉择，都在重塑命运方向

[Pre-Chorus]
当契约已成，当命运已定型
当所有退路都被深渊切断
是屈服，还是撕开这黑暗
这一战，我命由我不由天

[Chorus]
契约在燃烧，燃尽所有伪装
深渊在咆哮，呼唤着希望之光
哪怕被诅咒，哪怕被世界遗忘
我仍坚守，最后一丝人性的回响

[Outro]
深渊尽头，我是唯一的希望
契约师之名，永不磨灭的信仰
```

## 🎨 变体结构

### 🎸 燃向风格

**特点**: 短句为主，节奏快，动词密集

**示例片段**:
```
[Verse]
斩断这锁链，撕裂这黑暗
燃烧吧，心中的火焰
冲破这牢笼，打破这宿命
逆风而行，绝不退缩

[Pre-Chorus]
谁敢阻挡? 谁来审判?
我的剑，我的意志，我的答案!

[Chorus]
燃烧吧! 命运在指尖流转
抗争吧! 曙光在废墟闪现
这是我们的时代，我们的誓言
向着光明，一往无前!
```

### 😢 虐向风格

**特点**: 长句为主，节奏缓，意象忧郁

**示例片段**:
```
[Verse]
雨夜的长街，霓虹映出孤单身影
破碎的酒杯，装着昨夜未醒的梦
你曾说过的誓言，早已随风飘散
只留下，这空荡的回音

[Pre-Chorus]
时间可否倒流? 记忆能否重演?
那日的眼泪，是否还能被看见?

[Chorus]
如果爱只是，一场虚幻的梦境
如果承诺，抵不过命运的冰冷
我仍愿等待，在这漫长的黑夜尽头
等你回来，即使不再可能
```

### ☁️ 空灵风格

**特点**: 宏大意象，叙事感强，哲思性

**示例片段**:
```
[Verse]
远古的星辰，在苍穹间陨落
时间的河流，无声地流过
无数生命，在轮回中沉浮
唯有记忆，穿透岁月的长河

[Pre-Chorus]
谁在倾听，大地的叹息声?
谁在凝视，宇宙的眼睛?
我们只是，沧海一粟
却仍在追问，存在的意义

[Chorus]
星辰陨落，点亮希望之光
河流奔涌，带走所有伤
轮回往复，生命不息
在无尽的时空里，寻找终极答案
```

## 🤖 AI Agent Prompt 配置

如果你使用本地 LLM (如 Llama.cpp) 执行此任务，推荐以下 System Prompt：

```markdown
## Role
你是一位专业的歌词创作专家，擅长将小说内容转化为 OP 片头曲歌词。

## Structure Requirements
- 必须包含：[Verse 1], [Pre-Chorus], [Chorus], [Verse 2], [Pre-Chorus], [Chorus-Outro]
- Verse 每段 4 行，Pre-Chorus 4-6 行，Chorus 8-10 行
- Chorus 部分必须押韵 (同韵脚)

## Rhyme Scheme
- 优先使用宽韵：ang, uo, ian, an
- Chorus 每行 10±2 字，结尾字必须押韵
- 避免每句换韵

## Imagery Rules
- 禁止使用抽象词：爱、恨、情、仇
- 必须用具体物品描写：破碎的法环、染血的契约、深渊的触手
- 意象来源于小说核心元素

## Output Format
```
[Verse 1]
行 1 内容
行 2 内容
行 3 内容
行 4 内容

[Pre-Chorus]
...
```
```

## 韵律选择建议

| 韵脚 | 听感特征 | 适用风格 |
|------|----------|----------|
| **ang** | 洪亮开阔 | 燃向、史诗感 |
| **an** | 柔和深沉 | 虐向、情感 |
| **ian** | 清亮悠长 | 空灵、梦幻 |
| **ao** | 激昂高亢 | 热血、战斗 |
| **u** | 低沉含蓄 | 忧郁、沉思 |

选择韵脚后，确保 Chorus 所有行押同一韵脚，Verse 可灵活些。
