# 韵律与可唱性指南

## 宽韵选择

宽韵是指发音相近、可以互相押韵的韵脚组合。相比严格押韵，宽韵在可唱性上更灵活，听众感知为"押韵感"而非"刻意押韵"。

### 推荐韵脚组合

| 韵母 | 代表字 | 听感特征 | 最佳适用场景 |
|------|--------|------|----|
| **ang** | 光、亡、往、望、芒、荒、疆 | 洪亮、开阔、有力 | 燃向、史诗战斗、高潮段落 |
| **an** | 暗、岸、散、叹、念、远、变 | 深沉、柔和、忧郁 | 虐向、情感段落、回忆场景 |
| **ian** | 念、念、见、变、远、陷、愿 | 清亮、悠长、梦幻 | 空灵风格、内心独白 |
| **ao** | 傲、涛、啸、跑、到、好、逃 | 激昂、高亢、决绝 | 热血战斗、宣言场景 |
| **u** | 路、苦、独、宿、续、负、悟 | 低沉、含蓄、内敛 | 忧郁、沉思、黑暗氛围 |
| **i** | 意、离、忆、泣、寂、力 | 尖锐、细腻、紧张 | 压抑、冲突场景 |
| **ou** | 头、守、后、朽、流、旧、久 | 绵长、哀婉、怀旧 | 离别、追忆、情感段落 |
| **ong** | 空、终、梦、中、痛、重、涌 | 浑厚、宏大、深沉 | 史诗感、宇宙观描写 |

### 宽韵组合表

以下韵母可互相押韵 (同一"韵部"):

```
【江阳韵】ang, ang, uang, iang
推荐字：光、芒、荒、疆、望、狂、亮、翔、荡、荡

【言前韵】an, ian, uan, üan
推荐字：岸、散、念、远、变、现、见、陷、愿、念

【遥条韵】ao, iao
推荐字：傲、涛、啸、逃、好、晓、飘、绕、倒

【姑苏韵】u, ü
推荐字：路、苦、独、宿、续、负、悟、雾、渡、顾

【由求韵】ou, iu
推荐字：头、守、后、朽、流、旧、久、候、透

【中东韵】ong, iong
推荐字：空、终、梦、中、痛、重、涌、虹、雄、踪
```

## 🎵 Chorus 押韵规则

Chorus 是歌词的灵魂，必须严格押韵以确保可唱性和记忆点。

### 基础规则

1. **同韵到底**: Chorus 所有行末尾字必须押同一韵
2. **推荐字数**: 每行 10±2 字 (8-12 字)
3. **避免换韵**: 不要在这一句押 ang，下一句突然变成 an
4. **自然优先**: 韵脚服务于内容，不要为了押韵而牺牲意义

### 示例分析

**✅ 正确示例 (ang 韵)**:
```
逆转这宿命的荒_唐_ (tang2)
在废墟之上寻找光_芒_ (mang2)
哪怕世界已走向灭_亡_ (wang2)
契约的火焰仍在烧_烫_ (tang4)
```
**评分**: 完美押韵，听感统一，每行 10 字整齐

**❌ 错误示例 (换韵)**:
```
逆转这宿命的荒_唐_ (tang2)
在废墟之上寻找光_芒_ (mang2)
哪怕世界早已散_失_ (shi1)  ← 突然变成"失"韵
契约的火焰仍在发_亮_ (liang4)
```
**评分**: 第三行突然换韵破坏整体感

### 押韵检查清单

- [ ] Chorus 所有行末尾字押同一韵
- [ ] 每行字数在 8-12 字范围
- [ ] 韵脚不刻意、不勉强
- [ ] 没有同韵部混用 (ang/ong/an 混用)

## 🔊 节奏控制

### Verse (世界观铺陈)

**节奏特征**: 相对自由，句长变化

**建议**:
- 4 行结构，每行 8-12 字
- 长短句交替，制造呼吸感
- 示例：

````
```
[Verse 1]
月光_下_，法环碎裂成_光_尘__ (9)
染血的契约书，在掌心_发_烫_ (10)
街灯下，恶魔低语如旧_友_ (9)
林夜转身，背影被夜色_拉_长__(11)
```
````

**技巧**: 第一行短 → 第二行中 → 第四行长，形成递进

### Pre-Chorus (情绪攀升)

**节奏特征**: 逐渐加速，句长缩短

**建议**:
- 4-6 行，每行 6-10 字
- 每行逐渐缩短，制造紧张感
- 可用疑问句制造悬疑

**示例**:
````
```
[Pre-Chorus]
当力__量__即将吞噬最后的理__智__ (12)
人性在深渊边缘摇__摇__欲__坠__ (10)  ← 缩短
退路已断，唯有向__前__ (8)  ← 更短
谁__在__呼唤，哪__里__有答__案__ (8)  ← 疑问
```
````

### Chorus (主题爆发)

**节奏特征**: 规整有力，句长统一

**建议**:
- 8-10 行，每行 10±2 字
- 可分成 2 段 (每段 4-5 行)
- 严格押韵

**示例结构**:
````
```
[Chorus]
契__约__在__燃烧__，燃尽__所有__伪装__ (12)
深__渊__在__咆哮__，呼唤__着希望__之__光__ (12)
哪怕被诅咒，哪怕被世界__遗__忘__ (11)
我仍坚守，最后一丝人性的__回__响__ (11)
```
````

## 🎼 押韵检查工具

### 人工检查法

1. **标记韵脚**: 将每行末尾字写出
2. **判断发音**: 只读韵母部分，忽略声调
3. **判断是否同韵**: ang/ang 可以，ang/ong 不行

**示例**:
```
检查押韵：
- 唐 (tang2)
- 芒 (mang2)
- 亡 (wang2)
- 烫 (tang4)
判断：全部是 ang 韵 ✅
```

### AI Prompt 检查

如果让 LLM 帮你检查押韵，使用以下提示:

```
请检查以下 Chorus 部分的押韵情况:
[粘贴歌词]

要求:
1. 提取每行末尾字的韵母
2. 判断是否所有字押同一韵
3. 如果不押韵，指出哪些行不标准
4. 给出修改建议
```

## 🚫 常见押韵错误

### 1. 同字重复

```
错误:
契约在燃烧，燃尽所有伪装
契约的火焰，依然在那燃烧

问题: "燃烧"重复出现
```

**建议**: 同一段落内，押韵字不要重复

### 2. 强行押韵

```
错误:
为了寻找那未知的答案 (an)
我必须穿越这漫长的夜 (ye) ❌

问题: 第二行硬凑韵脚，读起来不自然
```

**建议**: 先保证内容准确，再考虑押韵

### 3. 韵部混淆

```
错误:
光 (guang1 - ang 韵)
空 (kong1 - ong 韵)
中 (zhong1 - ong 韵)

问题: "光"不能与"空"押韵
```

**建议**: 确认韵部后再押韵，不要混用

## 🎶 可唱性评分标准

| 标准 | 评分 | 说明 |
|------|------|----|
| **押韵一致性** | 严格押韵 (10 分) / 宽韵 (8 分) / 混韵 (3 分) | Chorus 必须严格押韵 |
| **字数对齐** | 每行 8-12 字整齐 (10 分) / 有长短变化 (7 分) / 参差不齐 (2 分) | 对齐更有节奏感 |
| **呼吸空间** | 每行中间有停顿 (10 分) / 无停顿 (5 分) | 避免一口气唱不完 |
| **语义连贯** | 语句通顺自然 (10 分) / 有小瑕疵 (7 分) / 生硬 (2 分) | 不为了押韵牺牲语义 |
| **意象密度** | 适中 (10 分) / 过密 (5 分) / 过稀 (5 分) | 避免像念经或太空洞 |

**总分 50 分**:
- **45-50 分**: 专业级，可唱性优秀
- **35-44 分**: 良好， minor 调整即可
- **25-34 分**: 需大改，主要问题在韵律和句式
- **25 分以下**: 不可唱，需重写

## 🎯 实践建议

### 新手起步

1. **从宽韵入手**: ang/an 韵最容易写
2. **固定字数**: 先练习每行固定 10 字
3. **押同一韵**: 不要混韵
4. **用词自然**: 不为了押韵编生硬语句

### 进阶技巧

1. **交替句式**: 8 字 + 12 字交替，制造节奏
2. **半押韵**: Verse 可宽松，Chorus 必须严格
3. **重复句**: 用重复句作为 anchor，增强记忆点
4. **结尾钩子**: Chorus 最后一句要特别亮眼
