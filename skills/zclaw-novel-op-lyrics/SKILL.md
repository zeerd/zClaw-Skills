# Novel OP Lyrics Assistant

专业的小说片头曲歌词创作工具——将文学性融入节奏感。

## 核心能力

根据小说内容或概要，创作结构化的 OP 歌词作品：

- **素材提取**: 自动提取核心意象、情绪基调、关键词
- **结构创作**: Verse (世界观铺陈) → Pre-Chorus (情绪攀升) → Chorus (主题爆发)
- **韵律控制**: 支持宽韵押韵 (ang/uo/ian 等)，确保可唱性
- **模板填空**: 提供可直接使用的歌词结构模板
- **音乐风格推荐**: 根据小说类型自动匹配适合的音乐风格，并提供详细的风格描述用于 AI 音乐生成参考
- **导出功能**: 自动将创作结果导出到 Google Sheets

## 创作流程

### 第一阶段：灵魂素材提取

在动笔之前，从小说中提取决定歌词风骨的核心要素：

1. **核心意象 (Visual Motifs)**
   - 选出 3-5 个代表性物品或场景
   - 例如：破碎的义体、染血的古剑、永不停歇的雨、黄昏的教室

2. **情绪基调 (Tone)**
   - 确定曲风方向
   - 燃向 (热血/快节奏)、虐向 (忧郁/慢节奏)、空灵 (宏大/叙事感)

3. **关键词 (Hook Words)**
   - 小说中最常出现的概念
   - 如"轮回"、"重构"、"契约"、"深渊"、"宿命"

4. **节奏参考**
   - 可选：指定一首现有歌曲作为节奏模板
   - 控制每句字数和整体节奏

### 第二阶段：音乐风格匹配与描述

根据小说类型和情绪基调，推荐并生成详细音乐风格描述：

1. **风格分类匹配**
   - **燃向**：流行摇滚、电子摇滚、交响金属
   - **虐向**：新灵魂乐、独立民谣、钢琴抒情、电子氛围
   - **空灵**：新世纪音乐、后摇、环境音乐、古典融合
   - **其他**：爵士乐、嘻哈/R&B、合成器 wave

2. **风格描述生成**
   - 包含：音色特征、节奏型态、情感氛围、适用场景
   - 示例：**新灵魂乐**——"一首温暖、充满原生态质感的曲目，洋溢着真实的乐器演奏和毫不费力的律动。现场鼓手敲打出一种宽松、受嘻哈影响的节奏型——柔和的底鼓带着慵懒的摇摆感，军鼓的击打总是稍微拖在节拍后面一点点，而刷击的踩镲则充满了呼吸感，伴随着人性化的不完美，沙沙作响。"

3. **输出格式**
   - **推荐风格**：[风格名称] 或 [多风格组合]
   - **风格描述**：可用于 AI 音乐生成工具的详细描述文本
   - **匹配理由**：基于小说元素说明推荐依据

### 第三阶段：三段论创作

#### 1️⃣ Verse (世界观铺陈)
- **内容**: 描写主角身处的环境或内心矛盾
- **技巧**: 特写镜头
- **示例**: 不写"世界很冷"，写"霓虹灯影里，铁锈的味道在扩散"

#### 2️⃣ Pre-Chorus (情绪攀升)
- **内容**: 节奏加快，预示改变或冲突到来
- **技巧**: 疑问句或动作短语制造迫切感
- **示例**: "指针倒流的瞬间，谁在深渊边缘呼唤？"

#### 3️⃣ Chorus (主题爆发)
- **内容**: 全曲核心，点明小说主旨
- **技巧**: 必须押韵且字数对齐
- **示例**: "逆转这宿命的荒唐，在废墟之上寻找光芒"

### ⚠️ 核心注意事项：避开"尬写"

| 维度 | 建议做法 | 避雷要点 |
|------|----------|----------|
| **韵律** | 尽量使用**宽韵** (ang/uo/ian)，唱起来更响亮 | 避免每句换韵脚或完全不押韵 |
| **词汇** | 使用具象动词 (撕裂/燃烧/折射) | 堆砌辞藻，使用过多无意义形容词 |
| **节奏** | 注意长短句交替，留出间奏空间 | 字数太密导致像念经 |
| **关联度** | 歌词要让读者联想到小说情节 | 写得太通用，可替换到任何小说 |

## 输入格式

### 最少输入 (自动提取)

```
小说标题：《某作品名》
小说概要：[300-500 字的故事梗概]
```

### 推荐输入 (完整参数)

```
小说标题：《某作品名》
小说概要：[故事梗概]

创作参数:
- 曲风：燃向/虐向/空灵 (默认：自动分析)
- 音乐风格：流行摇滚/新灵魂乐/电子摇滚/等 (推荐：自动匹配)
- 目标韵脚：ang/uo/ian/其他 or 自动检测 (默认：宽韵)
- 副歌每行字数：10±2 字 (默认：推荐值)
- 核心意象：[指定 3-5 个物品/场景，不填则自动提取]
- 参考曲风：参考歌曲名 (可选，用于节奏参考)
- 生成风格描述：是/否 (默认：是，用于 AI 音乐生成)
```

### AI Agent Prompt 模板

如果让 Llama.cpp 或其他本地 LLM 执行此任务，请在 Prompt 中加入以下约束：

```
## 结构约束
强制要求包含：[Verse-1], [Pre-Chorus], [Chorus], [Verse-2], [Pre-Chorus], [Chorus- Outro] 标签

## 韵律指令
请使用【ang】韵结尾，确保副歌每行 10 个字

## 意象指令
禁止使用"爱、恨、情、仇"等抽象词，必须通过具体物品描写情感，例如：
- ❌ "我恨这世界的冰冷"
- ✅ "寒铁剑在掌心发颤，锈蚀的锁链已崩断"
```

## 使用示例

### 最简单用法

```
请用此技能为小说写 OP 歌词：

《轮回重构师》
一个被遗忘的世界，主角在轮回中不断重构命运。
曲风：燃向，韵脚：ang
```

### 完整用法

```
请用 novel-op-lyrics 技能创作歌词：

小说：《深渊契约师》
概要：在这个魔法崩坏的时代，契约师们与深渊恶魔签订契约换取力量，却逐渐失去人性。主角林夜是最后一个坚持不背叛的契约师，他必须在力量与人性之间找到平衡，对抗失控的深渊洪流。

创作设定:
- 词风：虐向转燃向
- 音乐风格：新灵魂乐 (或：钢琴抒情/独立民谣)
- 韵脚：ian/an (宽韵)
- 核心意象：破碎的法环、深渊触手、染血的契约书、破碎的月
- 参考节奏：类似《火影忍者》ED 风格
- 副歌每行：10 字左右
- 生成风格描述：是
```

### 带音乐风格描述的用法

```
请用 novel-op-lyrics 技能创作歌词：

小说：《赛博黎明》
概要：未来都市中，人类与 AI 共同生活在虚拟与现实交织的世界，主角是一名突破虚拟界限的觉醒者。

创作设定:
- 词风：燃向
- 音乐风格：电子摇滚
- 韵脚：ian/an (宽韵)
- 生成风格描述：是

期待输出包含详细的电子摇滚风格描述，用于 AI 音乐生成工具的参考。
```

## 输出格式

每个歌词作品包含：

1. **📋 创作概览**
   - 小说标题与主题
   - 提取的核心要素 (意象/基调/关键词)
   - 韵律与节奏特征

2. **🎵 音乐风格匹配**
   - **推荐风格**：[风格名称] 或 [多风格组合]
   - **风格描述**：详细的声音描述文本 (可用于 AI 音乐生成工具)
   - **匹配理由**：基于小说元素说明推荐依据

3. **🎶 完整歌词**
   ```
   [Verse 1]
   [具体场景/物品描写]
   [主角动作/环境压抑感]

   [Pre-Chorus]
   [突变/心境转折]
   [准备战斗/启程心态]

   [Chorus]
   [核心口号 - 关于命运/反抗]
   [核心意象 - 关于希望/目标]
   [收尾句 - 扣回小说标题]
   ```

4. **📊 导出结果**
   自动将以下内容导出到 Google Sheets：
   - 小说元数据 (标题/主题/创作日期)
   - 歌词结构 (Verse/Pre-Chorus/Chorus 各段落)
   - 韵律分析 (韵脚类型/可唱性评分/意象密度)
   - 音乐风格信息 (推荐风格/风格描述文本/匹配理由)
