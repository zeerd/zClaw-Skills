# Article Recommender - 文章推荐序创作技能

https://weibo.com/1727858283/Q3hLEEQwV

这是一个用于为科技文章撰写高质量推荐序的技能。

## 功能

当你提供一篇科技文章时，该技能会：

1. **深度剖析**：理解文章核心论点、关键洞察和基调
2. **提炼金句**：识别 2-3 句最能代表文章灵魂的精彩语句
3. **创作推荐序**：以科技专栏主笔的身份，撰写一篇 400-600 字的独立推荐短文

## 使用方法

直接提供文章内容或链接，例如：

> 请用推荐序技能为我这篇文章写推荐序：[文章内容]

或

> 帮我给这篇文章写个推荐序：[文章链接]

## 输出特点

- ✅ 独立成篇的短篇评论，而非原文摘要
- ✅ 以观点驱动的论证结构
- ✅ 金句无痕融入行文
- ✅ 引人入胜的钩子开篇
- ✅ 升华价值的结尾号召
