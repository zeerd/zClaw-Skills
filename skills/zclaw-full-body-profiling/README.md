## 🚀 快速开始

当你需要创建一个完整的人物描述时，只需提供人物的基本特征，技能会自动生成结构化的全人描述。

### 输入示例

**基础信息**:
```
性别：男，年龄：30 岁左右，体型：运动型，身高：180cm
```

**衣着风格**:
```
穿着黑色连帽卫衣、灰色运动裤、白色运动鞋
```

**动态特征**:
```
走路较快，双手常插兜
```

---

## 💡 核心能力

1. **三维立体描述**: 静态骨架 → 动态身姿 → 衣着修饰
2. **结构化输出**: 清晰的模块化描述模板
3. **AIGC 优化**: 直接可用的英文提示词格式
4. **一致性保障**: 标志物锚点与分层生成策略

## 📋 使用示例

### 示例 1：刑侦嫌疑人力描述

**用户输入**:
```
需要描述一个 35 岁左右的男性嫌疑人，看起来中等身材，穿夹克和牛仔裤，走路时有点跛
```

**技能输出**:
```markdown
### 【整体印象】
35 岁男性，中胚层运动体型与内胚层混合，身高约 175cm，中等身材给人普通印象

### 【骨架细节】
肩宽：平均宽度，非突出特征
胸腹：中等厚度，无明显肌肉线条
四肢：右腿略短于左腿（跛行的物理证据）

### 【身姿与动态】
站姿：重心偏向左腿（减轻右脚压力）
手部习惯：自然下垂，无明显动作
步态特征：右腿轻微跛行，行走速度快度缓慢

### 【衣着与状态】
上装：深色夹克合身，表面无明显磨损
下装：蓝色牛仔裤，标准合身度
鞋履：深色运动鞋，右脚可能有磨损痕迹
整体风格：保守实用的日常着装
```

## 🎯 输出示例（完整 AIGC 提示词格式）

```markdown
**Subject**: Female, 30s, mesomorph athletic build, 170cm tall, broad shoulders, defined waist.
**Posture**: Standing upright, hands on hips, confident gaze.
**Attire**: Wearing fitted combat pants, armored vest over tactical shirt, combat boots.
**Style**: Militarystyle, practical, ready for action.
**Environment**: Urban combat zone, dusk lighting.
**Camera**: Full body shot, confident pose, 50mm lens, realistic texture, 8k.
```

## ⚡ 快速提示

- 提供**基础身份**（性别、年龄、体型）是最重要的
- **标志性物品**比面部特征更容易在多次生成中保持一致性
- **不完美的细节**（皱巴巴的衣服、磨损的鞋子）大大增加真实感
- 使用**分层生成**策略：先确定全身，再细化面部
