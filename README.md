# zClaw-Skills

* article-analyzer : 文章分析
* article-recommender ： 给文章做序
* paper-popularization ：专业文献“中译中”
* web-keywords-searcher ：基于必应搜索关键词
* web-content-fetcher : 获取网页内容
* prompt-to-image : 提示词生成图片（Comfy UI）
* article-to-cartoon-prompt ： 文章生成卡通画提示词
* article-to-cartoon-image : 文章卡通组画化
* weixin-article-analyzer ： 微信公众号文章解析

## 安装

在龙虾终端输入：

```bash
./install.sh
```
