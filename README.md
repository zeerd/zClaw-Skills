# zClaw-Skills

## Tool

* fetch_web_content : 获取网页内容

## Skill

### 文章解析

* zclaw-article-analyzer : 文章分析
* zclaw-article-recommender ： 给文章做序
* zclaw-article-to-cartoon-prompt ： 文章生成卡通画提示词
* zclaw-paper-popularization ：专业文献“中译中”
* zclaw-weixin-article-analyzer ： 微信公众号文章解析

### 视觉

* zclaw-consistent-face-prompt : 面部细节提示词生成
* zclaw-full-body-profiling ： 身形细节提示词生成
* zclaw-prompt-to-image : 提示词生成图片（Comfy UI）

### 其他

* zclaw-article-to-cartoon-image : 文章卡通组画化
* zclaw-web-keywords-searcher ：基于必应搜索关键词

## 安装

在龙虾终端输入：

```bash
./install.sh
```
